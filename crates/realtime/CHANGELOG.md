## [0.1.0] - 2026-02-21

### Added
- Initial release of the `rust-oxide-realtime` crate.
- Reusable realtime server runtime for Axum, including auth verification and channel policy hooks.
- Rust websocket client utilities for connecting, joining channels, and emitting events.
- `chat_demo` example showcasing end-to-end realtime flows.
